# Communication Simulator

Python 通訊模擬器

## Project Overview

本專案模擬數位通訊系統的基本流程：

Text
↓
Binary Encoding
↓
Communication Channel
↓
Bit Error Rate (BER)
↓
Receiver
↓
Statistics
